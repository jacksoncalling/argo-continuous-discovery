# Interview Snapshot Template

One page maximum. If it does not fit, cut narrative — keep decisions.

---

## Header

| Field | Value |
|-------|-------|
| **Date** | YYYY-MM-DD |
| **Interview #** | N of minimum 3-4 for this outcome |
| **Interview Quality** | Rich / Mixed / Thin (from 02-interview assessment) |
| **Interviewee** | Role, context — NO names or PII |
| **Segment** | Which customer segment (e.g., "Online gift buyer", "Spa manager", "First-time purchaser") |
| **Context** | 1-2 sentences on the interview setting and what was discussed |
| **Product Outcome** | The team's current desired outcome this interview maps to |

---

## Experience Map

The story the interviewee told, broken into sequential steps. Each step is a moment in their experience. Use the interviewee's own language where possible.

```
1. [First thing that happened] →
2. [Next thing] →
3. [The friction point or decision moment] →
4. [What they did as a result] →
5. [Where they ended up]
```
